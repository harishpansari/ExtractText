# ExtractText

A Text Extraction web application using Bootstrap and Tesseract.js. This is a Single Page Appliaction with client-side rendering.

This application extract text in 20+ international languages.

## Tech Stack
* HTML
* CSS
* JavaScript
* Bootstrap
* Tesseract.js
